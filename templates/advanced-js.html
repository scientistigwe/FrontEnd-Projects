2. Data Processing Task

Entrust the web worker with a demanding computational task as follows: 1. Data Loading: Load a sample dataset from a CSV file in the main script. The dataset MOST have "quantity" and "unit price" columns. 2. Total Revenue Calculation (Web Worker): Determine the total revenue for each entry by multiplying the "quantity" and "unit price" columns. 3. Price Statistics Computation (Web Worker): Compute the maximum and minimum values of the "quantity" and "unit price" columns. 4. Results Transmission: Transmit the total revenue, quantity and price statistics back to the main thread for visualization.
